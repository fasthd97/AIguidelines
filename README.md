AI Guidelines
A plain-language guide to using AI tools for your job search.

What This Is
This guide was written to help people who are new to AI get practical, immediate value from it during a job search. It is based on a real job search using these exact methods.
It covers:

Privacy and AI — what you need to know before you type anything
Which tool to use and an honest comparison of the main options
How to set rules so AI behaves consistently
How AI memory works and how to work around it
How to tailor a resume for each job without lying
How to write cover letters fast
How to track applications for unemployment benefit compliance
What to trust and what to verify yourself
Ready-to-use prompts you can copy right now
State unemployment tracking requirements
Free AI learning resources


Who This Is For
Anyone navigating a job search who wants to use AI effectively without getting burned by it.
Written with veterans in mind but applicable to any job seeker.

The One Rule Worth Knowing Before You Open It

Your confidence in AI output should scale with your ability to verify it — not with how confident the AI sounds.

AI presents correct answers and made-up answers with exactly the same voice. The guide explains how to tell the difference.
